# Screenshot Guide (Mandatory Submission)

Capture the following pages with white/light background after running the app:

1. Visitor input form

- URL: http://localhost:8080/request
- Include filled sample data before submit

2. Student approval screen

- URL: http://localhost:8080/approvals
- Show approve and reject buttons

3. Visitor logs

- URL: http://localhost:8080/logs
- Show status, entry time, and exit time

4. Reports dashboard

- URL: http://localhost:8080/reports
- Show summary counts

Store screenshots in a local folder named screenshots and include them in your PPT/report.
